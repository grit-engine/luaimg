Building from Source

The source code for LuaImg lives in the Grit svn repository. On windows, use the VC project file. On unix-based OS, use the following steps (makefiles may need to be tweaked if your system is weird):

  1. Build the lua-5.1.4 fork (type make in /dependencies/lua-5.1.4).
  2. Install you have readline and freeimage installed. On debian/ubuntu: libfreeimage-dev. OSX users report: brew install freeimage.
  3. Change to /luaimg and run make there.
  4. The luaimg executable should be in /luaimg ready for deployment.

There is more information on the Grit Compilation Instructions page, but note these instructions build the whole of Grit which you may not want to do. For community support, use the Grit IRC channel and/or forums.
