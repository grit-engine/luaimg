Download

Static builds of LuaImg are available for Windows and Linux (Ubuntu).

Building from Source

The source code for LuaImg lives in the Grit svn repository, and can be built by first building the lua-5.1.4 fork in the dependencies directory, and then building LuaImg. There are makefiles and visual studio project files for doing this. LuaImg depends on freeimage and (on linux only) libreadline. See the Grit Compilation Instructions for more information, and check into the Grit chat room or forums if you need help.
